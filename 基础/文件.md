# 文件操作

## 打开文件

```python
f = open("文件路径", "访问方式")
# 关闭文档
f.close()
```

## 文件访问方式

### r

read 只读，文件必须存在，否则会报错。

```python
# 只读
f = open("/Download/test.txt", "r")
str = f.read()
print str
# 关闭文档
f.close()
```

### w

write 只写，不能读。文件存在会覆盖内容，不存在则创建文件

```python
# 覆盖
f = open("/Download/test1.txt", "w")
f.write("我会覆盖文字")
# 关闭文档
f.close()
```

### a

append 追加，在文件末尾增加内容.文件不存在会自动创建

```python

# 追加写
f = open("/Download/test1.txt", "a")
f.write("我会在原有文字后增加")
# 关闭文档
f.close()
```
