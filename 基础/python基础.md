## 列表与切片

字符串和数组都可以使用切片选择特定的范围

```python
str[a:b]
numbers[a:b]

# 省略开头，开始与序列出事
numbers[:99]

# 省略结尾，结束与序列末尾
numbers[3:]

# 更大的步长，每隔多少个取一个
numbers[0:10:2]
```

## 字典

字典项的顺序不确定

### 字典方法

* clear 删除所有
* copy 返回新字典，是浅复制，替换副本中的值时，原件不受影响，deepcopy深拷贝
* fromkeys 创建新字典
* get 访问字典中的项
* items 返回所有字典项item饿列表，以(key，value)的形式
* keys 返回字典中的键
* pop 获取指定键的值，并从字典中删除该键值对
* popitem 随机删除一个字典项
* setDefault 可以添加新的键值对
* update 使用一个字典中的项更新另一个字典
* values 返回由字典中的值组成的字典视图
